Overview

At present, any changes from the original appearance of Makahiki are mostly cosmetic due to implementation difficulties, and the new widgets are not fully functional. The Makahiki documentation is still generally useful for configuring the system.

Profile page Changes

An article creation widget was partially implemented: it creates an entry in the database, but the resulting article has no text and the entry is not currently retrieved by any of the other widgets.

User Organization

Users are organized into normal users and administrators, consistent with the plans for the final implementation of the wiki. Users must be manually added by one of the existing administrators at the moment, so the system is currently only functional as an invite-only site.
