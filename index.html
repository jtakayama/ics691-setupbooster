SetupBooster

(ics691-setupbooster)

SetupBooster is a project that has the goal of creating a Django-based wiki that can teach new users how to use itself by completing demonstration activities while encouraging participation by awarding points and badges for user involvement and contributions. It is based on another Github project, Makahiki.
